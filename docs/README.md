# Skills Documentation Structure

This directory contains modular documentation for the Speed Reader Web App.

## Structure

The documentation is organized into three categories:

### Core
- **overview.md** — Project description, features, deployment, limitations
- **architecture.md** — File structure, storage, CORS proxy
- **dependencies.md** — Defuddle (npm), esbuild, Dependabot
- **orp.md** — ORP algorithm and word delay multipliers
- **shortcuts.md** — Keyboard shortcuts reference

### UI
- **screens.md** — Input, reader, finish screens, DOM elements, URL parameters

### Plans
- **roadmap.md** — Version history and planned features

## Usage

Start with [SKILLS.md](SKILLS.md) which serves as the main entry point with links to all modules.

Each module can be read independently and includes a link back to the main file.

## File References

The main SKILLS.md file uses the `#[[file:path]]` syntax to reference these modules, allowing for easy navigation and context inclusion in AI-assisted development tools.
