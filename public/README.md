# Public Assets

This folder contains static files served to the browser.

## Structure

- `index.html` - Main HTML page
- `favicons/` - App icons and manifest for different platforms

## Deployment

This folder is the root directory for deployment. Configure your hosting provider to serve files from the `public/` directory.

See deployment configurations:
- `netlify.toml` (Netlify)
- `vercel.json` (Vercel)
